Baldur's Gate 3

Local Save Editor

A local web tool for inspecting and experimenting with BG3 save files. It runs on the user's own PC so saves stay local.

Open GitHub Repo Download ZIP

What It Does

Save Discovery

Scans common Windows BG3 save folders and can also import a manually selected .lsv file.

Save Inspection

Shows party data, modules, save metadata, LevelCache entries, and editable extracted files.

Recovery Tools

Creates backups, restores mod order from a save, changes metadata fields, and repacks edited copies.

How To Run

  1. Download the ZIP from GitHub and extract it.
  2. Install Node.js 18 or newer.
  3. Install Norbyte/LSLib ExportTool so Divine.exe is available.
  4. Double-click START SERVER.bat.
  5. Open http://localhost:8081.
npm start

Safety

Original saves are not overwritten. Repacked BG3 saves can still trigger the game's tamper/corruption warning, especially after deep Globals.lsx changes. Always test edited copies first.
